# MQTT Message

## Installation
* Install paho-mqtt
```pip3 install paho-mqtt```
* You can run directly ```install.bat``` for quick installation

## Run
* You can run ```python3 main.py``` or using ```run.bat```

## Usage
* Before you can publish and receive messsage after subcribing, you need to click connect or subcribe respectively.
* After connecting, if sending message, you should see the published message on both internal (for only what you send) and external (for the message you receive after subcribe to **MQTT Broker**).
